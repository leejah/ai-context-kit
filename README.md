## ⚙️ Managing Different Context File Types

ai-context-kit supports several popular files for AI agents:

- **CLAUDE.md:** Contains instructions for the Claude AI agent.
- **AGENTS.md:** Lists multiple agents and their details.
- **Cursor rules:** Settings related to text cursors used by AI coding tools.
- **Copilot instructions:** Customized control data for GitHub Copilot.

You can open any of these files one by one. The tool will treat each properly, showing relevant information and checks for each type.

## 🗂 Understanding Results and Metrics

When you load a file, you may see several outputs:

- **Linting results:** These are suggestions or errors about problems in your file. Fixing them helps the AI understand your instructions better.
- **Context size:** This shows the length or amount of data in your file. Some tools have limits, so it helps to know how much you are using.
- **Compliance check:** Confirm your files meet expected formats or rules.
- **Summary view:** A simple overview to help you scan what is inside the files quickly.
